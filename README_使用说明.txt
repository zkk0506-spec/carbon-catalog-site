碳纤维产品在线目录网站 - 使用说明

1. 打开方式
   - 解压 carbon_catalog_site_updated.zip
   - 双击 carbon_catalog_site/index.html 即可在浏览器中预览

2. 文件结构
   - index.html：网站首页/产品目录页面
   - assets/images：产品目录图片、产品分类图、工艺图、应用图、生产设备图
   - assets/certificates：RoHS / MSDS 图片缩略图
   - assets/docs：原始 PDF 证书、报告、产品目录文件
   - assets/logo.png / logo-small.png / logo.webp：公司 Logo

3. 已更新内容
   - 使用真实上传素材替换原来网页里的示意图/占位图
   - 增加产品大类图片卡片
   - 增加 Product Range / Carbon Fiber Tubes / Sheets / Rods / Fabric / Raw Materials / Custom Parts 等完整图文模块
   - 增加 RoHS & MSDS 证书报告下载中心
   - 增加 RFQ 询盘表单和产品筛选表格

4. 正式建站建议
   - 可以把 assets 文件夹整体上传到网站服务器
   - index.html 可以拆分为 Home、Products、Tubes、Sheets、Materials、Custom Parts、Certificates、Contact 等多页面
   - RFQ 表单需要由建站人员连接邮箱、WhatsApp、CRM 或网站后台
